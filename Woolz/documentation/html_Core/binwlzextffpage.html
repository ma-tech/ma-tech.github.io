Data Format Conversion

Introduction

Command line binaries that involve reading or writing non-native woolz formats are collected together under the "External File Format" (ExtFF) heading. These binaries typically match procedures in the WlzExtFF library. Supported formats can be determined by running the programme WlzExtFFConvert with the flag "-h" and are:

Other programmes in this group convert a few special formats for contours and particualar applications. For conversion of raw data then use WlzRawToWlz in the extended applications module.
