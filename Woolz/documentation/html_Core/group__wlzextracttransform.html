WlzExtractTransform
Extended Woolz Applications.
Name
WlzExtractTransform - finds the affine transforms from a reconstruction bibfile.
Synopsis
WlzExtractTransform [-h] [-v] [-A] [-R] [-a] [-t] [-T] [-f<filename>]
Options
-h Prints usage information.
-v Version operation.
-A Calculate absolute transform.
-R Calculate relative transform, default.
-a Output and argument string for WlzAffineTransformObj, default.
-t Output a Woolz affine transform object.
-T Output text.
-f Filename to be found.
Description
WlzExtractTransform finds the transform for the given filename within the bibfile and outputs either an argument string for WlzAffineTransformObj, a Woolz affine transform or a text description.
